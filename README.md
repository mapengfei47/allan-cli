# Allan-cli

## 安装

~~~bash
npm install allan-cli -g
allan init templateName projectName
~~~



## 命令行

~~~bash
allan --help #查看使用帮助
allan -V --version #查看工具的版本号
allan list #列出所有可使用的模版
allan init <template-name> <project-name> #基础指定的模版进行项目初始化
~~~
